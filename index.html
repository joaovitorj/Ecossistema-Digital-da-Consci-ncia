Sobre

O que é este espaço, e como usá-lo.

Este é um espaço de reconhecimento — não de instrução. As perguntas não têm resposta certa; elas existem para que você perceba o que já está em você, antes de buscar fora.

Se você chegou aqui perguntando "como faço algo assim", a resposta começa por reconhecer: você não precisa de um manual para começar. Volte para o Reconhecimento quando quiser continuar.
